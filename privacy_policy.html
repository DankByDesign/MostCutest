Unity Analytics Identifiers

Lil Brother Games collects the following identifiers via Unity Analytics:

  • Unique device identifiers (e.g., IDFV for iOS devices and Android ID for Android devices)
  • IP address
  • Country of install (mapped from IP address)
  • Device manufacturer and model
  • Platform type (iOS, Android, Mac, Windows, etc.)
  • Operating system and version
  • Language
  • CPU information (model, number of CPUs, frequency, instruction set support flags)
  • Graphics card type and vendor name
  • Graphics card driver name and version
  • Graphics API in use (e.g., “OpenGL 2.1” or “Direct3D 9.0c”)
  • System and video RAM present
  • Current screen resolution
  • Version of the Unity Editor used to create the game
  • Sensor flags (e.g., device support for gyroscope, touch pressure, or accelerometer)
  • Application or bundle identification (“app ID”) of the game installed
  • Unique advertising identifiers provided for iOS and Android devices (e.g., IDFA or Android Ad ID)
  • Checksum of all data transmitted to verify correctness
